# Maria Naz — QA Portfolio

QA Engineer with 4+ years of manual testing experience across web, mobile, and RFID-based applications. This repository collects practical examples of my QA work: test documentation, automation with Playwright (Python), API testing with Postman, performance testing with JMeter, and CI/CD integration with GitHub Actions.

---

## 👋 About Me

I'm a QA Engineer based in Lahore, Pakistan. My experience includes:

- **Manual Testing (primary focus):** Functional, regression, usability, and end-to-end testing of browser and mobile applications
- **Test Automation:** Playwright with Python — using codegen and Page Object Model
- **Performance Testing:** JMeter load tests validating systems under 500+ concurrent users
- **API Verification:** Postman for request/response validation
- **Tools I use daily:** JIRA, Git, Postman, JMeter, Playwright

Currently working as an SQA Engineer at Innovent Tech, where I lead manual QA for an RFID tracking system deployed across 50+ facilities in the UAE.
